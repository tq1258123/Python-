# 注释

---

在写`Python`代码时，由于一个项目可能需要很久才能写完，或者一个项目需要和别人进行合作，所以为了能够更容易回忆自己以前写的代码功能，或者让合作者更容易理解自己写的代码，我们需要用注释来详细写清楚代码的用途。

```python
# 单行注释

'''
多行
注释
'''

"""
多行
注释
"""

def func(a, b):
    """
    函数注释：这个函数是用来计算a和b的和
    :param a: 第一个数据
    :param b: 第二个数据
    :return:  返回两个数之和
    """
    return a + b

print(func(2,3))
```
