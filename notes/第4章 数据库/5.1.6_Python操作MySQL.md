## 创建数据库

```mysql
create database if not exists mytestdb character set utf8;
use mytestdb;
drop table if exists `employee`;
create table `employee` (
`id` int(11) not null auto_increment,
`name` varchar(40) default null,
`age` int default null,
`sex` char(1) default null,
`income` float default null,
primary key (`id`)) engine=InnoDB default charset=utf8;
```

## 数据库插入操作

```python
import pymysql
# 打开数据库连接
db = pymysql.connect(host="192.168.0.105", user="root", password="582153",
                    database="mytestdb", port=3306)
# 使用cursor()方法获取操作游标
cursor = db.cursor()
# SQL插入预计
sql = "insert into employee(name, age, sex, income) values('%s', '%d', '%s', '%d')" % ('王五1', 25, 'F', 5000)
try:
    # 执行sql语句
    cursor.execute(sql)
    # 提交到数据库执行
    db.commit()
except Exception as e:
    print(e)
    db.rollback()
finally:
    # 关闭数据库
    db.close()
```

在数据库里插入单条数据时，使用的数据格式`'%s', '%d', '%s', '%d'`

## 数据库批量插入操作

```python
import pymysql
# 打开数据库连接
db = pymysql.connect(host="192.168.0.105", user="root", password="582153",
                    database="mytestdb", port=3306)
# 使用cursor()方法获取操作游标
cursor = db.cursor()
# SQL插入预计
sql = "insert into employee(name, age, sex, income) values(%s, %s, %s, %s)"
ls = []
employee1 = ('张三', 22, 'F', 2000)
employee2 = ('李四', 23, 'M', 3000)
ls.append(employee1)
ls.append(employee2)
try:
    # 执行sql语句
    cursor.executemany(sql, ls)
    # 提交到数据库执行
    db.commit()
except Exception as e:
    print(e)
    db.rollback()
finally:
    # 关闭数据库
    db.close()
```

在数据库里插入多条数据时，使用的数据格式`%s, %s, %s, %s`

## 数据库查询操作

`fetchone/fetchall/fetchmany`

```python
import pymysql
# 打开数据库连接
db = pymysql.connect(host="192.168.0.105", user="root", password="582153",
                    database="mytestdb", port=3306)
# 使用cursor()方法获取操作游标
cursor = db.cursor()
# SQL插入预计
sql = "select * from employee where income > %d" % (2000)
try:
    # 执行sql语句
    cursor.execute(sql)
    # 获取所有记录列表
    results = cursor.fetchall()
    for row in results:
        id = row[0]
        name = row[1]
        age = row[2]
        sex = row[3]
        income = row[4]
        # 打印结果
        print("id=%s, name=%s, age=%d, sex=%s, income=%d" % (id, name, age, sex, income))
except Exception as e:
    print(e)
finally:
    # 关闭数据库
    db.close()
```

数据库查询操作时，注意查询内容以元组形式`(id, name, age, sex, income)`
